# Blog-project-for-SSU
Sample project of blog web site for Sumy state university

### Technology stack:
- Java 17
- Spring Boot 3
- Spring Security 6
- Spring Web
- Spring Data
- Lombok
- PostgreSQL
- HTML, CSS
- Thymeleaf
